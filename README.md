Skill Swap is a full-stack web application built using Flask, MySQL, and Jinja2 that allows students to showcase their skills, find collaborators, and exchange knowledge for academic projects, hackathons, and competitions.

The platform focuses on peer-to-peer collaboration, making it easier for students to connect based on skills, interests, and project requirements.

➡️Key Features

User Authentication

Secure registration and login using hashed passwords (bcrypt)

Email-based authentication system

User Profiles

Display skills, purpose, contact details, and profile information

Search and filter users based on skills or interests

🔎 Search & Filter

Keyword-based search across name, skills, and purpos

Filter users based on collaboration interests

💬 Messaging System

One-to-one messaging between users

Mutual messaging report using SQL joins

🤝 Collaboration Requests

Post project collaboration requests

View collaboration opportunities posted by other users

🔔 Notifications

User notifications with read/unread status

📊 Advanced SQL Usage

Joins, correlated subqueries, grouping, and reporting queries

Practical implementation of DBMS concepts

🛠️ Tech Stack

Backend: Flask (Python)

Frontend: HTML, CSS, Jinja2 Templates

Database: MySQL

Security: bcrypt for password hashing

Server: Gunicorn (deployment ready)

This project was developed to:

Apply DBMS concepts in a real-world application

Practice full-stack web development using Flask

Enable meaningful student collaboration and skill sharing

It is especially useful for:

College students

Hackathon teams

Peer learning and mentorship
